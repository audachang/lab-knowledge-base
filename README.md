# ACL Lab Knowledge Base

A structured knowledge base for the ACL lab (audachang / aclexp). Contains fixes, configurations, and workflows — primarily for use by AI agents but also useful for humans.

## Structure

```
fixes/          Software bugs and environment fixes
servers/        Server-specific notes and configurations
workflows/      Common task workflows
```

## How to Use (for AI agents)

Fetch any document directly:
```
https://raw.githubusercontent.com/audachang/lab-knowledge-base/main/<path>
```

Fetch the index to find relevant documents:
```
https://raw.githubusercontent.com/audachang/lab-knowledge-base/main/INDEX.md
```
